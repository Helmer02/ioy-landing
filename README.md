# IOY Landing Page

Landing page institucional desenvolvida para apresentar a IOY, empresa focada em soluções digitais como sistemas SaaS, automações e ferramentas para pequenos negócios.

## 🚀 Sobre o projeto
A IOY Landing foi criada com o objetivo de comunicar de forma clara e profissional os produtos e soluções oferecidos pela empresa, com foco em conversão, experiência do usuário e apresentação comercial.

## 🎯 Objetivo
Criar uma página moderna, responsiva e otimizada para apresentar a marca, seus serviços e captar novos clientes.

## ⚙️ Funcionalidades
- Apresentação da empresa e soluções
- Estrutura focada em conversão (CTA)
- Layout responsivo
- Organização visual moderna
- Base preparada para expansão

## 🛠 Tecnologias utilizadas
- Next.js
- TypeScript
- Tailwind CSS

## 🧠 Aprendizados aplicados
- Estruturação de landing page profissional
- Organização de layout e componentes
- Melhoria de experiência do usuário
- Construção de interface com foco comercial
- Padronização visual de produto digital

## 📌 Status do projeto
Em desenvolvimento contínuo
